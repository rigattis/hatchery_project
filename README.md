# ReadMe:

This project was developed as part of the Software Engineering course at Boston College. This project was developed in collaboration with Tianli Qu, Hengrun Zhao, Matthew Lim, and Alejandro Gonzalez. 

The app has been deployed using Render and is available at: https://hatchery-ibdi.onrender.com/

Additionally, the app can be run locally by following the subsequent steps:

1. clone the repo from github
2. naviagate to the hatchery_project directory using the command line
3. create a local virtual environment
- python -m venv venv (mac)
- source venv/bin/activate (mac)
or
- python -m venv venv (windows)
- venv\Scripts\activate (windows)
5. install requirements: pip install -r requirements.txt
6. navigate to the hatchery folder: cd hatchery
7. run the command files:
- python manage.py add_contacts
- python manage.py add_events
- python manage.py add_machines
- python manage.py add_projects
- python manage.py add_schedule
- python manage.py add_spaces
- python manage.py add_trainers
6. run the app locally: python manage.py runserver
7. navigate to the app on localhost
